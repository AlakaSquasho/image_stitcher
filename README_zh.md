# Image Stitcher

一个轻量的单文件图片拼接工具，直接在浏览器中打开即可使用，无需安装依赖或启动服务。

[English README](./README.md)

## 功能

- 支持横向拼接、纵向拼接、2x2 拼接、3x3 拼接
- 横向与纵向模板支持自定义图片数量
- 支持拖拽导入、点击选择图片、粘贴剪贴板中的图片
- 自动填充可用槽位，导出时按格子居中裁剪
- 支持调整画布显示比例
- 支持调整图片间距，自定义分隔颜色与宽度
- 支持导出 PNG、JPG、WEBP
- 支持浅色 / 深色主题切换
- 支持快捷键导出与切换主题

## 使用方式

1. 直接在浏览器中打开 `index.html`
2. 在左侧选择拼接模板
3. 将图片拖入画布，或点击槽位选择图片
4. 在右侧工具栏调整图片数量、显示比例、图片间距和导出格式
5. 点击“导出图片”保存结果

## 交互说明

- 横向拼接 / 纵向拼接：可自定义图片数量
- 2x2 拼接：固定 4 张图片
- 3x3 拼接：固定 9 张图片
- 可以直接粘贴剪贴板中的图片到当前空槽位
- 快捷键：
  - `Ctrl/Cmd + E`：导出图片
  - `Ctrl/Cmd + D`：切换主题

## 适用场景

- 快速把多张图片拼成一张长图或网格图
- 制作简单的对比图、截图合集或九宫格
- 不想引入构建工具，只想要一个可直接打开的本地工具
